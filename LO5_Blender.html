Learning Blender

Description

My efforts to learn Blender by watching tutorials in YouTube

What I Did

This is the first time I ever touched Blender and I've always wanted to try to make some 3D things. I wanted to be a horror game designer so that means I want to be in Game Design specialization later in semester 4.

The first time I used Blender, so ofcourse I looked into some YouTube tutorials and I found a popular tutorials from Blender Guru. The tutorials showed how to make a donut but I don't want to just blindly following him so I decided to make a tart cake instead.

So far, I really liked sculpting and I found myself enjoying it so much. Unfortunately, lately I didn't have any time to continue my cake even though I really want to start learning about coloring, texturing and more.

Other than watching YouTube, I also joined Blender workshops at school. Currently we are learning on how to make a kitchen diorama, and it was fun and not too complex and intimidating for me so I really enjoyed it.
