Portfolio Javascript

Description

I wanted to incorporate some JavaScript into my portfolio, so I decided to research and follow tutorials on how to create a functioning navigation bar and image carousel.

What I Did

I started by searching for online resources and tutorials to learn how to implement a navigation bar and image carousel using JavaScript. I found valuable references from W3Schools and used explanations from ChatGPT to understand the logic behind creating an image carousel. After studying these resources, I wrote JavaScript code for the carousel images, which helped me grasp the overall concept. Unfortunately, I had to delete the image carousel part because the teachers found it difficult to check my portfolio with this feature. I replaced it with something simpler, but the process gave me valuable insights into JavaScript.

Reflection

How it went: Although I had to remove the image carousel due to usability issues for the teachers, the experience was still valuable. I successfully implemented the navigation bar and learned a lot about JavaScript through this process.

What I learned: I gained a solid understanding of the logic behind JavaScript for image carousels and enhanced my coding skills. This experience taught me the importance of considering usability and simplicity in design.

What will change: In the future, I will focus on creating simpler, more user-friendly features and test them thoroughly to ensure they are easy to use for all stakeholders. Additionally, I will continue to build on my JavaScript knowledge to implement more advanced features when appropriate.
