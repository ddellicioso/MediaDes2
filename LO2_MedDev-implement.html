Project 3 Website Development

Description

For the third and last group project, we were asked to fix fontys ICT’s website and implement them. My teammates and I made some iterations of Figma prototype of our ideas on our version of fontys website before actually implementing them.

What I Did

I was tasked to make the profile page (media page), so after receiving answers from the survey and did some iterations regarding the page, I started making the HTML and CSS. Fortunately, my teammates already made the navigation and footer section, so I just needed to focus on making the profile page.

The responses from the survey really helped a lot because I didn’t need to think about how I should organize the content since the respondents already told me what to do as well as which design I should make.

The hardest part to make was creating the carousel image for the job opportunity section. I did make something similar for my portfolio (previously) but the mechanic is a little bit different. Instead of showing the images one by one, I needed to show 3 images at once and when the next/previous button is clicked, the carousel moves only the most left/right image. After the struggle, I finally succeeded in making it functional even though it’s a little bit different from the prototype but I did my best.

Reflection

How it went: I was responsible for the profile page (media design). Using the survey responses as a guide, I started building the HTML and CSS for the page. The survey feedback was incredibly helpful because it gave me a clear direction on how to organize the content and which design elements to include.

The toughest part was creating the image carousel for the job opportunities section. Unlike my previous portfolio carousel that showed images one by one, this one needed to display three images at once and move only one image left or right with each click. After struggling and searching through tutorials, I finally got it to work. While it ended up a bit different from the prototype, I did my best to make it functional.

What I learned: The survey responses were valuable. They saved me a lot of guesswork and ensured the design met user expectations. Creating a more complex carousel taught me a lot about advanced CSS and JavaScript techniques. It was challenging but rewarding to see it work.

What will change: Based on the feedback, I’ll make sure that elements that look clickable actually are clickable to avoid confusion. For projects and external links, I’ll make sure they open in new tabs to improve user experience. Overall, this project was a great learning experience. It showed me the importance of user feedback, the complexities of advanced web features, and the value of teamwork.
