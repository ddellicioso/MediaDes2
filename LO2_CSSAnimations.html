CSS Animations

Description

Before adding any animation, my portfolio felt boring and not interactive enough. I decided that adding simple animations would make it more dynamic and engaging .

What I Did

Initially, I thought that my portfolio lacked interactivity and felt static. This led me to the idea that incorporating animations could enhance its visual appeal and make it more engaging for users. To gain the necessary skills, I attended CSS lectures given by one of the teachers, Metaxas, which provided valuable insights into creating animations.

Using the concepts learned from these lectures, I set out to add simple animations to the portfolio. I focused on implementing hover effects, transitions, and keyframes animation to various elements like button, images, and text. This process invloved experimenting with different animation styles and durations to find the most effective ways to create a dynamic and interactive experience.


Reflection

How it went: Adding animations made the portfolio look more dynamic and engaging, significantly improving its interactivity and overall appeal.

What did I learn: Creating animations taught me how animation works in web design, enhancing my understanding and boosting my confidence is using CSS for animations.

What will change: In the future, I will incorporate animations earlier in the design process and explore more advanced animation techniques to further enhance interactivity and user experience.
